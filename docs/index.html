스와이프 지식 도서관

저장만 하던 습관에서 읽는 습관으로

간편한 저장

공유 버튼 한 번이면 어떤 앱에서든 링크를 바로 저장합니다. YouTube, Instagram, 블로그, GitHub 등 플랫폼을 자동으로 인식합니다.

스와이프로 읽기

읽은 링크는 오른쪽, 나중에 볼 링크는 왼쪽으로 스와이프. 밀린 콘텐츠를 게임하듯 빠르게 소화하세요.

라벨로 정리

나만의 라벨로 링크를 분류하세요. 라벨별 읽기 진행률을 한눈에 확인할 수 있습니다.

읽기 알림

라벨별로 요일과 시간을 설정하면 읽지 않은 링크가 있을 때 알림을 보내드립니다.

나만의 도서관

북마크, 메모, 읽음 상태 관리까지. 저장한 링크를 체계적으로 관리하세요.

클라우드 동기화 (선택)

Google 또는 Apple 계정으로 로그인하면 여러 기기에서 데이터를 동기화할 수 있습니다. 로그인 없이도 모든 기능을 그대로 사용할 수 있습니다.

개인정보 처리방침

시행일: 2026년 4월 8일

1. 수집하는 정보

클립(Clib)은 기본적으로 로그인 없이 사용할 수 있으며, 이 경우 개인정보를 수집하지 않습니다. 모든 데이터는 사용자의 기기에만 저장됩니다.

클라우드 동기화를 위해 Google 또는 Apple 계정으로 로그인하는 경우, 다음 정보가 수집됩니다:

• 계정 이메일 주소 (사용자 식별 목적)
• 사용자 고유 ID (Firebase Authentication)
• 사용자가 저장한 아티클 정보 (URL, 제목, 썸네일 URL, 플랫폼, 라벨, 읽음/북마크 상태, 메모)
• 사용자가 생성한 라벨 정보 (이름, 색상)

2. 정보의 이용 목적

수집된 정보는 오직 기기 간 데이터 동기화 목적으로만 사용됩니다. 마케팅, 프로파일링, 제3자 제공 등의 목적으로는 사용되지 않습니다.

3. 정보의 저장 및 보호

로그인한 사용자의 데이터는 Google Firebase(Cloud Firestore)에 암호화되어 저장됩니다. Firebase의 보안 정책은 Firebase 개인정보처리방침을 참고하십시오.

4. 제3자 서비스

• Firebase Authentication / Cloud Firestore — 로그인 및 데이터 동기화 (개인정보처리방침)

• Google AdMob — 광고 표시 (개인정보처리방침)

5. 데이터 삭제

사용자는 앱 내 설정 > 계정 삭제를 통해 언제든지 계정을 삭제할 수 있습니다. 계정 삭제 시 Firebase에 저장된 모든 클라우드 데이터가 영구적으로 삭제됩니다. 기기에 저장된 로컬 데이터는 유지됩니다.

6. 웹페이지 접속

앱은 저장된 링크의 메타데이터(제목, 썸네일)를 가져오기 위해 해당 웹페이지에 접속합니다.

7. 아동의 개인정보

본 앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

8. 개인정보 처리방침 변경
