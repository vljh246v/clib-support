Clib

스와이프 지식 도서관

저장만 하던 습관에서 읽는 습관으로

간편한 저장

공유 버튼 한 번이면 어떤 앱에서든 링크를 바로 저장합니다. YouTube, Instagram, 블로그, GitHub 등 플랫폼을 자동으로 인식합니다.

스와이프로 읽기

읽은 링크는 오른쪽, 나중에 볼 링크는 왼쪽으로 스와이프. 밀린 콘텐츠를 게임하듯 빠르게 소화하세요.

라벨로 정리

나만의 라벨로 링크를 분류하세요. 라벨별 읽기 진행률을 한눈에 확인할 수 있습니다.

읽기 알림

라벨별로 요일과 시간을 설정하면 읽지 않은 링크가 있을 때 알림을 보내드립니다.

나만의 도서관

북마크, 메모, 읽음 상태 관리까지. 저장한 링크를 체계적으로 관리하세요.

개인정보 처리방침

클립(Clib)은 사용자의 개인정보를 수집하지 않습니다.

회원가입이 없으며, 모든 데이터는 사용자의 기기에만 저장됩니다. 서버로 전송되는 데이터는 없습니다.

앱은 저장된 링크의 메타데이터(제목, 썸네일)를 가져오기 위해 해당 웹페이지에 접속하며, 이 외의 네트워크 통신은 하지 않습니다.

광고 표시를 위해 Google AdMob SDK를 사용하며, 이에 따른 데이터 처리는 Google 개인정보처리방침을 따릅니다.
